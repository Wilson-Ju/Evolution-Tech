# Turoriel MARKDOWN

`Ceci est ma toute première applicatioin du markdown`

Pour écrire deux paragraphes, mettre **un espace** entre les *2 paragraphes*

Pour un *titre de 1er niveau, mettre **un #**, pour un titre de second niveau, mettre **##**, etc...*

Pour mettre en italic, on met une étoile avant et après l'expression : *Bonjour*

Pour mettre en gras, mettre deux étoile : **Bonjour**

Pour des listes : mettre un tiret, et pour imbriquer des listes, tabuler avant de mettre le tiret:
- Fruits
    - Tomates
    - Ananas
- Légumes:
    - Tomates
    - Poirreau

Pour mettre une citation, par exemple repondre à une personne, mettre un chevron ouvrant :

> Salut à toi developpeur !
> Je sais que 
> jdfd

Pour un séparateur horizontal : mettre 3 tirets
---
Pour mettre en avant une emphase du code, mettre l'expression dans des bactics:
`Bonjour à tout le monde`

Si on veut montrer du code :  mettre 3 bactics les uns à la suite de l'autre :

```php
<?php

    class Mytest{
        public function stat(int $arg) {
            ...
        }
    }
?>

```
